1. < html >
    Description: The root element of an HTML document. It encapsulates all other elements and defines the document type as HTML.
  2. < head >
    Description: Contains meta-information about the HTML document, such as title, links to stylesheets, scripts, and other metadata.
  3. < title >
    Description: Sets the title of the HTML document, which is displayed in the browser's title bar or tab
  4. < body >
    Description: Contains the visible content of the HTML document, including text, images, links, and other elements displayed on the web page.
  5. < h1 > to < h6 >
    Description: Represents six levels of headings. < h1 > is the highest level, indicating the main heading, while < h6 > is the lowest level.
  6. < p >
    Description: Represents a paragraph of text. It is used to structure and group blocks of text within the document.
  7. < a >
    Description: Creates a hyperlink, allowing users to navigate to another web page or resource when clicked.
  8. < img >
    Description: Embeds an image in the document. It requires a src attribute that specifies the image URL.
  9. < ul >
    Description: Represents an unordered (bulleted) list of items. Each list item is represented by the < li > tag.
  10. < ol >
    Description: Represents an ordered (numbered) list of items. Each list item is represented by the < li > tag.
  11. < li >
    Description: Represents an individual item in a list (< ul > or < ol >). It can contain text, images, or other elements.
  12. < div >
    Description: A block-level container used to group elements and apply styles or JavaScript behavior to them.
  13. < span >
    Description: An inline container used to apply styles or scripting to a specific portion of text or other inline elements.
  14. < table >
    Description: Represents tabular data. It consists of rows (< tr >), which contain cells (< td >) or header cells (< th >).
  15. < form >
    Description: Creates a form for user input. It contains various form elements like input fields, checkboxes, buttons, etc.
  16. < input >
    Description: Represents an input field that allows users to enter data. It can be of different types, such as text, password, checkbox, etc
  17. < button >
    Description: Creates a clickable button that can trigger actions or submit form data.
  18. < textarea >
    Description: Represents a multiline text input field, allowing users to enter larger blocks of text.
  19. < select >
    Description: Creates a dropdown list from which users can select one or more options.
  20. < iframe >
    Description: Embeds another HTML document or web page within the current document.
