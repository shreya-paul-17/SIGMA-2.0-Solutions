What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create the structure and content of web pages. HTML uses a system of tags to define the different elements and their relationships within a document.

What is Semantic HTML?

Semantic HTML refers to using HTML elements that convey meaning and describe the content they enclose. It makes the document structure more meaningful to both developers and web browsers, improving accessibility and search engine optimization (SEO).

Common Semantic Tags:

  • <header>: Represents the header section of a document or a section.
  • <nav>: Represents a group of navigation links.
  • <main>: Represents the main content of a document.
  • <article>: Represents a self-contained piece of content that could be distributed and reused independently.
  • <section>: Represents a thematic grouping of content.
  • <aside>: Represents content that is tangentially related to the content around it, such as a sidebar.
  • <footer>: Represents the footer section of a document or a section.
  • <figure>: Represents any content that is referenced from the main content, such as images or diagrams.
  • <figcaption>: Represents a caption or legend for a <figure> element.
